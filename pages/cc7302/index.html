# 2026年4月学习总结

提示

本文记录了2026年4月的学习内容、成果与思考。

# 学习内容

本月主要专注于以下几个方面的学习：

  1. 计算机专业课

    • Java基础：this关键字、getter/setter、面向对象基础
    • C语言：基础知识点复习
    • 电路与模拟电子技术：电路计算、稳态电路分析

  2. 公共基础课

    • 高等数学：多元函数微积分、隐函数求导、二重积分与三重积分
    • 大学物理：牛顿定律、功和能、电路分析、动量与冲量
    • 大学英语：单词积累、阅读、翻译、听力与口语练习
    • 思政类课程：中国近代史、习近平新时代中国特色社会主义思想概论

  3. 其他实践学习

    • 非遗植物拓染制作
    • 课程课题讨论与考试复习

# 技术难点突破

本月高频难点攻克方向 高数：多元函数偏导、梯度、隐函数求导与重积分计算 物理：电路综合大题与动量守恒应用 英语：汉译英翻译与阅读判断类题型

# 每日笔记摘要

# 4月30日

英语阅读判断两篇学习，35个单词学习，电路与模拟电子技术期中考试卷

# 4月23日

高数多元函数复习，汉译英句子两句+阅读，交流稳态电路分析学习。

# 下月计划

  • 深入推进高数重积分后续章节的学习与刷题
  • 巩固物理电路与动量相关知识点，突破综合大题
  • 加强英语翻译与听力专项训练
  • 完成C语言、Java后续课程的预习与复习
  • 推进思政课程相关学习任务

# 思考与反思

# 本月在学习过程中，我发现自己往往倾向于按科目分散完成每日任务。在下个月的学习中，我会更注重知识体系的梳理，平衡基础刷题和综合应用练习，同时优化每日学习计划，提高学习效率。
