# 2026年6月学习总结

提示

本文记录了2026年6月的学习内容、成果与思考。

# 学习内容

本月主要专注于以下几个方面的学习：

  1. 高等数学

    • 级数收敛发散基础概念，正项级数审敛法
    • 交错级数，绝对收敛，幂级数收敛半径与展开求和
    • 傅里叶级数学习，曲线积分与曲面积分错题复盘
    • 期末综合模拟卷刷题，高数公式定理梳理

  2. 大学物理

    • 静电场，热学错题复盘
    • 晶体管放大电路，电磁学基础计算题训练
    • 电路与模拟电子综合大题，物理期末模拟卷练习

  3. 大学英语

    • 每日单词背诵，音标跟读，听力专项训练
    • 汉译英句子，英语段落朗读，写作模板练习
    • 阅读刷题，口语跟读，整套四级真题训练

  4. 计算机专业课

  • Java程序设计知识点复习，实验报告撰写整理
  • SQL数据库基础语句练习，前端基础回顾

# 学习成果

# 课程进展

  • 完整完成级数，傅里叶级数章节全部知识点学习，配套习题全部完成，梳理高数积分+级数全套错题
  • 系统巩固大学物理静电场，热学，晶体管电路，电磁学内容，完成物理期末模拟卷训练
  • 持续每日英语输入输出训练，覆盖听力，单词，阅读，翻译，写作，口语全模块，完成多套四级真题训练
  • 完成Java实验报告收尾，复习SQL数据库基础，巩固前端网页骨架知识

# 每日笔记摘要

# 6月7日

高数回顾级数基础概念并完成5道基础习题；英语听力练习半小时、背诵四级单词并完成一篇阅读；复盘5道静电场物理错题。

# 6月14日

完成级数章节专项小测，针对薄弱题型查漏补缺；独立完成一篇英语作文；完成家教备课并开展线下授课。

# 6月23日

限时完成高数期末模拟卷，主攻积分与级数核心考点；英语听力与翻译专项练习；梳理物理电磁学、电路部分期末重点知识。

# 6月30日

对整月学习内容做整体复盘，梳理未掌握的薄弱板块；重温高数积分、级数高频公式考点；进行英语听力与口语跟读练习，维持语言学习语感。

# 下月计划

  • 四级系统冲刺：整套真题限时模考，强化翻译、写作两大提分模块，每日稳定背诵四级核心词汇，专项攻克听力短板
  • 前端知识系统复习：巩固HTML/CSS基础，系统重学JavaScript核心语法，练习页面布局案例，夯实网页开发底层能力
  • 大一计算机全学年复盘：梳理C语言、Java、数据库SQL、高数、电路电子全部核心知识点，搭建完整知识框架，整理全套错题集，完成综合自测查漏补缺
  • 突破大学物理电磁感应综合大题，过完物理全册期末考点，完成整套物理模拟试卷训练

# 思考与反思

# 本月各科刷题量充足，但缺少定期复盘，知识点体系零散。下月将减少堆积式刷题，每周定时梳理知识框架，平衡多科目学习节奏。
