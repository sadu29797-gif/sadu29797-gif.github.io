# 2025年12月学习总结

提示

本文记录了2025年12月的学习内容、成果与思考。

# 学习内容

本月主要专注于以下几个方面的学习：

  1. 英语学习

    • 单词积累：累计背诵单词310个，复习单词1890个
    • 专项练习：完成英语试卷、英语泛读/精读/视听说学习、雅思写作/口语/听力训练、英语文章翻译与熟读
    • 输出练习：翻译英语短文、口语练习40分钟、听力练习60分钟

  2. 数学学习

    • 高等数学：学习洛必达法则、泰勒公式、函数单调性与曲线凹凸性、定积分（概念/性质/公式/换元法/分部积分法）、有理函数积分、反常积分、定积分的几何/物理应用、微分方程（基本概念/可分离变量）
    • 线性代数：学习向量组及其线性组合/相关性/秩、向量空间、线性方程组解的结构、向量的内积/长度及正交性、线性空间的定义与性质/维数/基与坐标/基变换与坐标变换；完成相关习题与错题重做

  3. 编程与计算机学习

    • 代码编写：累计写二十三串代码，完成C语言基础知识题、高级语言程序（第一至六章、第九章指针）学习
    • 专业练习：完成20道计算机导论选择题、扣子平台开发实践实验

# 技术难点突破

在学习过程中遇到并解决了几个技术难点：

#include <stdio.h>
#include <stdlib.h>

int main() {
    int n = 5;

    int *arr = (int *)malloc(n * sizeof(int));
    if (arr == NULL) { // 判空避免野指针
        printf("内存分配失败\n");
        return 1;
    }

    for (int i = 0; i < n; i++) {
        arr[i] = i * 2;
        printf("%d ", arr[i]);
    }

    free(arr);
    arr = NULL;
    return 0;
}

#include <stdio.h>

void swap(int *x, int *y) {
    int temp = *x;
    *x = *y;
    *y = temp;
}

int add(int a, int b) {
    return a + b;
}

int main() {
    int num1 = 5, num2 = 10;
    swap(&num1, &num2);
    printf("交换后：num1=%d, num2=%d\n", num1, num2);

    int (*func_ptr)(int, int) = add;
    int result = func_ptr(3, 7);
    printf("3+7=%d\n", result);
    return 0;
}

# 每日笔记摘要

# 12月1日

完成大学英语泛读、视听说基础训练，刷C语言入门基础题，编写3段基础代码，同步背诵+复习英语单词，筑牢当月学习的英语与编程入门基础。

# 12月5日

主攻高等数学课后习题，巩固第三章核心知识点，完成线性代数42道基础练习题，搭配英语精读短文训练，通过刷题夯实数学学科入门根基。

# 12月12日

深入学习高等数学定积分概念、性质及计算方法，搭配1篇英语泛读材料拓展阅读量，实现数学核心知识点与英语阅读能力同步推进。

# 12月22日

复盘高级语言程序设计第一至六章内容，编写4段基础代码巩固语法，完成英语视听说进阶练习与精读训练，同步强化编程基础与英语应用能力。

# 12月29日

攻坚高级语言程序设计第九章指针难点，编写指针相关基础代码验证逻辑，开展60分钟英语听力专项练习，复习205个英语单词，实现编程难点突破与英语听力巩固。

# 下月计划

  • 编程学习：吃透高级语言程序设计结构体知识点，完成10道指针+结构体综合基础编程题，每道题至少2种解题思路；整理当月编程错题，标注错误原因。
  • 数学学习：梳理线性代数全章节基础概念，制作简易知识点卡片；每天做5道高数基础题+5道线代基础题，完成错题复盘。
  • 英语学习：每日背诵30个新单词+复习50个旧单词；每周完成2篇英语精读+3次30分钟听力练习；尝试写3篇100词左右的英语小短文。

# 思考与反思

# 本月在学习过程中，编程练习多为基础代码片段，缺乏知识点串联的小案例训练，下月会调整学习节奏，编程学习加入“知识点串联小案例”训练。
