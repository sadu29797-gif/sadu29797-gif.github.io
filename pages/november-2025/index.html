# 2025年11月学习总结

提示

本文记录了2025年11月的学习内容、成果与思考。

# 学习内容

本月主要专注于以下几个方面的学习：

  1. 英语学习

    • 单词背诵与复习（累计背诵超500个，复习超1500个）
    • 口语练习，听力训练及阅读/写作练习
    • 固定搭配，核心句型积累与汉译英训练

  2. 数学学习

    • 高等数学（函数连续性，导数，微分中值定理等章节学习+习题？错题练习）
    • 线性代数（矩阵，线性方程组等章节学习+习题/错题/试卷练习）

  3. 编程学习

    • C语言（输入输出函数，运算符与表达式，分支/循环控制语句，函数，数组）
    • CodeBlocks工具使用及代码实现

# 技术难点突破

在学习过程中遇到并解决了几个技术难点：

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>

int Factorial_Error(int n) {
    if (n >= 1) {
        return n * Factorial_Error(n - 1);
    }
    return 1;
}

int Factorial_Correct(int n) {
    if (n < 0) {
        printf("错误：n不能为负数\n");
        return -1;
    }
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * Factorial_Correct(n - 1);
}

int Factorial_Iteration(int n) {
    if (n < 0) {
        printf("错误：n不能为负数\n");
        return -1;
    }
    int result = 1;
    for (int i = 2; i <= n; i++) {
        result *= i;
    }
    return result;
}

int main() {
    int test1 = Factorial_Correct(5);
    if (test1 != -1) {
        printf("5的阶乘（递归）：%d\n", test1);
    }

    int test2 = Factorial_Iteration(10);
    if (test2 != -1) {
        printf("10的阶乘（迭代）：%d\n", test2);
    }

    Factorial_Correct(-3);

    return 0;
}

# 每日笔记摘要

# 11月4日

今天完成了多科学习任务：英语方面背诵了7-个新单词，复习175个旧词，积累了三个固定搭配和两个核心句型，还写了100词英语短文；数学完成部分习题，推进线性代数习题二的练习。

# 11月13日

今天重点在高数还有C语言：高数学习了第二章导数概念，求导法则及高阶导数的内容；C语言通过CodeBlocks实践，写了四串代码；英语则保持日常节奏。

# 11月27日

今天兼顾了英语，数学和编程：大学英语泛读学到第六课时，线性代数完成了一张试卷的练习；C语言继续实践写了四串代码。

# 下月计划

  • 完成英语泛读7到10课时，每周一篇阅读（找主旨句，长难句拆分），每周一次汉译英练习
  • 学习C语言指针，字符串处理，结构体，共用体和枚举。
  • 结束高等数学第三章的内容，学习线性代数第四章内容。
