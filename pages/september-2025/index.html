2025年10月学习总结

# 2025年9月学习总结

提示

本文记录了2025年10月的学习内容、成果与思考。

# 学习内容

本月主要专注于以下几个方面的学习：

  1. 英语学习

    • 背诵单词
    • 口语练习
    • 学习英语语法
    • 英语阅读能力

  2. 高数学习

    • 函数与极限

  3. 线性代数

    • 行列式
    • 矩阵及其运算

# 学习成果

  • 上英语口语课从完全听不懂到能够简单回答问题
  • 高等数学，线性代数学习过的课后习题都已掌握

# 项目进展

# 每日笔记摘要

# 10月3日

背了35个单词，学习了名词剩下的部分（名词的句法功能，名词和介词的搭配）还有代词（人称代词，物主代词，反身代词，指示代词，疑问代词），学习英语视听说教程第一单元的发音还有对话1和2，预习了函数并完成了习题。

# 10月22日

单词背诵35个，复习105个单词，口语练习60分钟，高等数学（无穷大与无穷小，极限运算法则），英语视听说倾听探索新闻报道，通道一，codeblock安装配置。

# 10月29日

背诵单词35个，复习单词105个，口语练习60分钟，学习识别同义替换（数字比例，近义词反义词，同根词，句式和逻辑语义的同义替换）和习题练习，学习高等数学第一章地六，七节（极限存在准则，两个重要极限，无穷小的比较）。

# 下月计划

  • 学习听力的基础语音知识(连读，省读，弱读)
  • 学习完高等数学函数与极限，导数和微分。
  • 背诵60个固定搭配，12个核心句型。
  • 四级词汇再背一遍。
  • 线性代数前三章学完。

# 思考与反思

# 时间安排不是不合理，感觉一个月下来很忙但是不知道在忙什么，可能这个月睡得有点晚精神也不是很好。目标不是很明确，感觉学习了但是又看不到学了什么。下个月要早点休息保持良好的状态，每天设置待办，知道自己忙什么。
